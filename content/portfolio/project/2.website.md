## Website

The very website you are on is a small project to maintain a pulse on modern web development. The site is built with [Nuxt](https://nuxt.com/) and [Nuxt UI](https://ui.nuxt.com/) which provides a solid foundation for building maintainable and accessible websites. The content is managed with [Nuxt Content](https://content.nuxtjs.org/) which allows me to write in Markdown and have it rendered without any hassle.

The entire site is statically generated via [Github Pages](https://docs.github.com/en/pages) and can be found on [GitHub](https://github.com/Behemyth/norrlander) itself.
